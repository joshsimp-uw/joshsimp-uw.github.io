Networking Tetris — VLSM Block Boundary Grid

Enter a base network (IPv4 or IPv6 CIDR). The grid represents the whole base block, split into 64 “pixels”. Drag a block from Block selection onto the grid, or tap a block and then tap the grid on mobile. Placement snaps to valid VLSM boundaries and blocks cannot overlap.

Examples: 10.0.0.0/20, 192.168.1.0/24, 2001:db8::/48, fd00:1234::/64
